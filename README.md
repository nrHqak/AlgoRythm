# AlgoRythm

AlgoRythm — это веб-платформа для изучения алгоритмов, которая позволяет студенту вставить собственный Python-код, отследить его выполнение по шагам, визуализировать работу алгоритма и получить обучающие подсказки от AI-ментора.

## Идея проекта

Основная проблема при изучении алгоритмов заключается в том, что студент видит только исходный код, но не видит, как именно он исполняется. AlgoRythm решает эту проблему за счёт пошагового трейсинга, визуализации массива в виде bar chart, аудио-сигналов событий и системы обучающих подсказок.

Пользователь может:

- запускать собственные алгоритмы на Python
- видеть пошаговый trace выполнения
- наблюдать анимацию сравнения, поиска, swap, shift и ошибок
- получать AI-подсказки без прямой выдачи готового решения
- проходить учебный путь по алгоритмам
- накапливать XP, достижения и стрики
- использовать ежедневные mini-quiz и Algorithm Pet для удержания привычки

## Использование AST

В проекте используется AST-анализ Python-кода для определения типа алгоритма.

Из кода извлекаются структурные признаки:

- количество циклов `for` и `while`
- глубина вложенности
- количество сравнений
- наличие swap через tuple assignment
- наличие `break`
- количество subscript-операций
- использование `range(len(...))`
- наличие floor division

На основе этих признаков модель классифицирует алгоритм как один из поддерживаемых типов:

- `bubble_sort`
- `selection_sort`
- `insertion_sort`
- `binary_search`
- `linear_search`

## Основные пользовательские сценарии

1. Пользователь регистрируется или входит в систему.
2. Открывает `Learn` и видит учебный путь.
3. Запускает ежедневный ritual:
   - кормит питомца
   - проходит 3 mini-quiz
4. Переходит в playground.
5. Вставляет Python-код алгоритма.
6. Получает:
   - trace
   - визуализацию
   - звук
   - тип алгоритма
   - оценку сложности
7. При ошибке получает подсказку от AI-ментора.
8. Результат сохраняется в профиль и влияет на XP, achievements и analytics.
